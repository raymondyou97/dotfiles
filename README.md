# dotfiles

My Personal Dotfiles

## Installation

Currently only works on Ubuntu/Debian. Repo must also be in home directory.

1. Install libs/dependencies

`./setup.sh -a`

2. Set up vim

`./setup.sh -v`

3. Set up rest (zsh, tmux, ssh)

`./setup.sh -r`
